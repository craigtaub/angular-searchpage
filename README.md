#AgularJS Doctify Search page
My angularjs app of a search page.

## Getting started
```
npm install
```

## Start the app
```
npm start
```
App is now running at `http://localhost:8000/app`

## Run the unit tests.
```
npm test
```

## Run the End-to-End tests
Setup the webdriver:
```
npm run update-webdriver
```
Run the tests:
```
npm run protractor
```

## SASS via compass
```
gem install compass
cd app/css
compass compile
```
